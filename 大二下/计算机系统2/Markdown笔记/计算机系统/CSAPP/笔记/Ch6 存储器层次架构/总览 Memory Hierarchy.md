# 学习存储的原因
**==存储器层次结构对应用程序的性能有着巨大的影响==**
**计算机系统中一个基本而持久的思想**：
如果你理解了系统是如何将数据在存储器层次结构中上上下下移动的，那么你就可以编写自己的应用程序，使得它们的数据存储在层次结构中较高的地方，使得CPU更快地访问到它们
